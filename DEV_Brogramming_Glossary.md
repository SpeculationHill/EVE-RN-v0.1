# DEV_Brogramming_Glossary

## Purpose
Provide a shared vocabulary for contributors working on EVE‑RN. Using consistent terminology helps avoid confusion and fosters a friendly brogrammer culture.

## Terms

- **Brogramming**: A light‑hearted term for coding and collaborating with camaraderie. Emphasises humour and support over elitism.
- **Capsuleer**: A player character in EVE Online, immortal due to cloning technology; the focal point of the game’s narrative.
- **New Eden**: The fictional universe where EVE Online takes place.
- **Empires**: The four major political entities – Amarr Empire, Caldari State, Gallente Federation, Minmatar Republic【414028640612414†L10-L16】.
- **NPSI (Not Purple Shoot It)**: A PvP doctrine where a fleet will engage any target not in their fleet regardless of allegiance【976944329353990†L113-L117】.
- **Highsec/Lowsec/Nullsec**: System security classes defined by their security level (0.5–1.0 for highsec, 0.1–0.4 for lowsec, –1 to 0 for nullsec)【482387685162354†L121-L125】.
- **Wormhole Space (W-Space)**: Uncharted systems connected by temporary wormholes【647673664077612†L154-L162】.
- **Faucet/Sink**: Economic terms describing mechanisms that create or destroy ISK in the game【940571151247159†L31-L38】【940571151247159†L83-L101】.
- **ESI (EVE Swagger Interface)**: The official RESTful API for EVE Online third‑party development【119277652475160†L55-L59】.
- **Archetype**: A defined player persona used to tailor recommendations (see B1_Archetypes_Ontology).
- **Blue/Purple**: In fleet chat, friendly pilots appear as blue or purple; “Not Blue, Shoot It” (NBSI) is a common null‑sec engagement rule.
- **YC**: “Yoiul Convention” dating system used in EVE lore (similar to CE years).
- **Bro Code**: The set of guidelines in the Contrib Playbook (DEV_Contrib_Playbook).

## Usage
- Contributors should use these terms in code comments, documentation and discussions to maintain consistency.
